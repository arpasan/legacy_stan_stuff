# CPS extract

`cps_cepr_org_2019_march.csv` is a slim extract from the
[CEPR CPS ORG Uniform Extracts](https://ceprdata.org/cps-uniform-data-extracts/cps-outgoing-rotation-group/cps-org-data/)
(Version 2.5), March 2019, ages 16–64.

Citation: Center for Economic and Policy Research. 2020. CPS ORG Uniform Extracts, Version 2.5. Washington, DC.

Columns: `year`, `month`, `rw`, `female`, `wbho` (Non-White indicator), `married`, `age`, `age_dec`.

The original 2020 write-up used a similar CEPR-style CPS extract (exact vintage unavailable); summaries will not match the legacy HTML digit-for-digit, but the model and workflow do.
